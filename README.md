# Amazon_Data_Analysis_EDA
This project focuses on analyzing Amazon sales data to uncover meaningful business insights using Python. The objective was to understand sales performance, revenue trends, customer behavior, and product-level performance through structured data analysis and visualization.

Tools & Technologies
Utilized Python-based data analytics techniques with Pandas and NumPy for efficient data handling, and Matplotlib and Seaborn to visualize insights that support data-driven business decisions. 

Key Analysis Performed
Data cleaning and preprocessing to handle missing and inconsistent values
Exploratory Data Analysis (EDA) to identify sales trends and patterns
Revenue and product performance analysis
Visualization of key metrics using charts and graphs for better interpretation

Key Insights
After the complete analysis performed we can see that significant customers from the Bengaluru city and Maharashtra state. Almost 99.9% product were sold out from the amazon itself. And demanding product were T-Shirts and 'M' size was preferred choice of the customers.
